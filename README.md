# 🎬 Sakuga Station

A sophisticated, cozy, and performance-optimized anime blog and content discovery engine built using Next.js and Tailwind CSS. The design blends modern Scandinavian minimalism with high-end cafe aesthetics, featuring fluid asymmetrical content blocks and fully responsive user layouts.

---

## 🎨 Design Language & Theme

Sakuga Station utilizes an organic, muted palette that provides high readability without the clinical starkness of pure black and white.

### Color Palette Reference
* **Background (`#F9F6F0`)**: A warm, soothing off-white cream canvas base.
* **Secondary (`#EFE5D5`)**: A soft, sandy beige used for navigation panels, active items, and card surfaces.
* **Foreground (`#2C302E`)**: A rich, deep charcoal-olive providing natural, accessible contrast for typography headers.
* **Accent (`#9CAF88`)**: A beautiful, muted sage green reserved for primary category highlights, dynamic tags, and call-to-actions.
* **Muted (`#827C73`)**: A stone taupe-gray ideal for metadata, disabled states, and comment descriptions.

### Typography Hierarchy
* **Headings (`font-josefin`)**: *Josefin Sans* — Used to elevate titles, display cards, logos, and critical navigation blocks with distinct character.
* **Body & Prose (`font-lato`)**: *Lato* — The default text standard chosen to ensure high visual comfort during long-form article skims.
